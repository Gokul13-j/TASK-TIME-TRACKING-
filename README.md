# Task & Time Tracking System (Beginner MERN)

Simple MERN project based only on your provided PDF requirements:
- JWT Authentication (Register/Login)
- Task CRUD (Create, Read, Update status, Delete)
- Time tracking (Start/Stop timer per task)
- Dashboard (Total tasks, completed tasks, total time)

## Project Structure

- `frontend/` - React + React Router + Axios
- `backend/` - Node.js + Express + MongoDB + Mongoose + JWT + bcrypt

## 1) Backend Setup

1. Open terminal in `backend`
2. Install packages:
   - `npm install`
3. Create `.env` from `.env.example`
4. Start backend:
   - `npm run dev`

## 2) Frontend Setup

1. Open terminal in `frontend`
2. Install packages:
   - `npm install`
3. Start frontend:
   - `npm run dev`

## Default URLs

- Frontend: `http://localhost:5173`
- Backend: `http://localhost:5001`

## MongoDB Collections Used

- Users
- Tasks
- TimeLogs
